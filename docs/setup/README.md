# SecureApps Setup Guide

Follow these steps in order to set up SecureApps.

## Setup Steps

| Step | Description | Time |
|------|-------------|------|
| [Step 1](step-1-prerequisites.md) | Prerequisites | 5 min |
| [Step 2](step-2-clone-and-install.md) | Clone and Install | 5 min |
| [Step 3](step-3-create-supabase-project.md) | Create Supabase Project | 5 min |
| [Step 4](step-4-environment-setup.md) | Environment Setup | 2 min |
| [Step 5](step-5-database-setup.md) | Database Setup | 10 min |
| [Step 6](step-6-create-admin-user.md) | Create Admin User | 3 min |
| [Step 7](step-7-run-application.md) | Run the Application | 2 min |

**Total estimated time: ~30 minutes**

## Quick Start

If you're experienced with Supabase and just need the SQL:

1. Clone repo and run `npm install`
2. Create `.env` with your Supabase credentials
3. Run the SQL from [Step 5](step-5-database-setup.md) in Supabase SQL Editor
4. Create admin user in Supabase Auth
5. Run `npm run dev`
